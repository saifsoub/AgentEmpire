AgentEmpire Secret Intake

Self-contained browser-only page for encrypting environment values before they are placed in the private project area. This page makes no network requests.

Do not commit plaintext secrets, encrypted payloads, or passphrases to git. Share the passphrase through a separate channel.

Checklist

  1. Open this page over HTTPS or locally from a trusted checkout.
  2. Encrypt only the variables needed for the target environment.
  3. Upload the encrypted JSON to the private project area.
  4. Send the passphrase through a different channel.
  5. After import, run npm run readiness, npm run typecheck, and npm run test.
